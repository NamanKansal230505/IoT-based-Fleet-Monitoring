      About this portal

      Krishi Yantra Track is a pilot platform for monitoring agricultural vehicles disbursed under state and central subsidy schemes. On-board telematics units transmit GPS coordinates and engine-state data over secure channels for compliance and utilisation review.

      This is a development build. Live data is shown for the Ghaziabad pilot unit; other districts display simulated data for demonstration.
